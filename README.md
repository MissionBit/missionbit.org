# Mission Bit Website

## Usage

Run the development server:

```bash
npm run dev
```

Fix any formatting or lint issues:

```bash
npm run fix
```

Run a full build of the site:

```bash
npm run build
```

Run the static export of the site:

```bash
npm run export
```
